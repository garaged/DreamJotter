# DreamJotter Privacy Statement

DreamJotter is a local-first macOS application.

## Project data

DreamJotter stores projects in `.dreamjotter` packages at locations selected by the user. Normal writing, editing, saving, opening, backup, restore, and export workflows do not require an account or a DreamJotter-operated cloud service.

DreamJotter does not upload screenplay content, project metadata, diagnostics, or usage telemetry.

## Support diagnostics

Support diagnostics are created only after the user explicitly selects **Export Support Diagnostics** and chooses a destination. The default report includes:

- DreamJotter version and build
- macOS version
- processor architecture
- current locale
- whether a saved project is open
- an optional bounded error summary

The default report does not contain screenplay text. Users should review the generated JSON file before sharing it.

## Exports and third-party storage

Exports and backups are written only to destinations selected by the user. Files saved to iCloud Drive, Dropbox, network storage, source-control systems, email, or other third-party services are subject to those services' privacy practices.

## Changes

Material privacy changes must be documented in release notes and reflected in both this file and the in-app privacy statement before release.
