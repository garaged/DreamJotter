# M13 Manual Validation Checklist

- [ ] Undo and redo typing, Smart Enter, Tab cycling, paste, cut, and autocomplete.
- [ ] Save preserves current-session undo; reopen starts a fresh undo session.
- [ ] Up/Down changes the active suggestion with TextKit focused.
- [ ] Return and Tab accept the active suggestion without requiring a click.
- [ ] Escape dismisses suggestions without editing text.
- [ ] Combined cues such as `SOFÍA / TOM` autocomplete and print correctly.
- [ ] VoiceOver announces paragraph type and active suggestion.
- [ ] Increased Contrast and larger text remain legible.
- [ ] Accented input, input methods, emoji, and large scripts remain stable.
