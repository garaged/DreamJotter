# Traceability Matrix

This matrix is the implementation navigation surface for DreamJotter. It maps product requirements and milestone feature areas to specs, acceptance documents, data contracts, executable spec coverage, planned modules, status, and notes.

Status values:

- `specified`: documentation exists and is ready to drive future executable specs or implementation planning.
- `executable-spec-pending`: documentation exists, but behavior-specific executable specs still need to be written.
- `implementation-pending`: executable specs or implementation can be planned next.
- `deferred`: intentionally out of scope for Milestone 1 through Milestone 4.
