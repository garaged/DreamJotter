# User Personas

## Beginner Writer

A beginner writer wants to start a screenplay without learning formatting rules first.

Goals:

- Start a new screenplay quickly.
- Write scenes, action, characters, and dialogue with minimal decisions.
- Export a readable script.
- Avoid technical configuration.

Mode fit: Simple Mode by default.

## Independent Filmmaker

An independent filmmaker writes and revises scripts while thinking about scenes, locations, characters, and production constraints.

Goals:

- Move through scenes quickly.
- Keep notes attached to meaningful screenplay elements.
- Export drafts for collaborators.
- Preserve local control of project files.

Mode fit: Simple Mode with selected Pro Mode controls.

## Professional Screenwriter

A professional screenwriter needs precise formatting, revisions, exports, and document control.

Goals:

- Control screenplay formatting and export behavior.
- Manage revisions and notes without corrupting script structure.
- Use semantic navigation and search.
- Customize workflows where supported.

Mode fit: Pro Mode.

## Technical Power User

A technical power user wants automation and extensibility, but is not the primary MVP driver.

Goals:

- Define repeatable commands or routines.
- Inspect portable project data.
- Eventually extend behavior through plugin APIs.

Mode fit: Pro Mode. Plugin support is future work and must not drive MVP architecture.
